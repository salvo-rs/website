# Compressione delle Risposte

Middleware per la compressione dei contenuti delle `Response`.

Fornisce supporto per tre formati di compressione: `br`, `gzip`, `deflate`. È possibile configurare la priorità di ciascun metodo di compressione in base alle esigenze.

## Compressione dei Dati nel Protocollo HTTP

1. A livello di formato file: tipi di file specifici utilizzano algoritmi di compressione dedicati
2. A livello di protocollo HTTP: implementazione della compressione delle risorse end-to-end tramite la codifica dei contenuti
3. A livello di connessione: compressione dei dati tra i nodi della connessione HTTP

Per ulteriori informazioni sulla compressione HTTP, consultare la [documentazione di Mozilla Developer Network](https://developer.mozilla.org/it/docs/Web/HTTP/Guides/Compression).
