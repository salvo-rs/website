# Middleware de Limitação de Tamanho de Upload de Arquivos

import { Tab, Tabs } from '@rspress/core/theme';

Um middleware para limitar o tamanho de arquivos enviados em requisições.

## Casos de Uso

O Limitador de Tamanho é utilizado principalmente nos seguintes cenários:
- Impedir que usuários enviem arquivos excessivamente grandes que consumam recursos do servidor
- Evitar estouro de memória do servidor causado por atacantes maliciosos que enviam grandes volumes de dados
- Definir políticas de limite de tamanho de requisição diferentes para diferentes endpoints de API
- Melhorar a eficiência e estabilidade do processamento de requisições do servidor
