提供對請求上傳檔案大小限制的中介軟體。

## 應用場景

Size Limiter 主要用於以下場景：
- 防止使用者上傳過大的檔案佔用伺服器資源
- 避免惡意攻擊者透過上傳大量資料導致伺服器記憶體溢位
- 為不同的 API 端點設定不同的請求大小限制策略
- 提升伺服器處理請求的效率和穩定性
