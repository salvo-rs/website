# Request-Chain-ID

import { Tab, Tabs } from '@rspress/core/theme';

Die Request-ID-Middleware ist äußerst flexibel. Der ID-Generator (IdGenerator) dient zur Erzeugung von IDs, und Sie können Ihren eigenen ID-Generator definieren, solange er das `IdGenerator`-Trait implementiert. Der standardmäßig bereitgestellte Generator ist `UlidGenerator`.

Darüber hinaus können Sie steuern, ob eine bestehende `requestid` überschrieben werden soll. Sie können auch `header_name` und andere Konfigurationen festlegen. Einzelheiten finden Sie in der [Dokumentation](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
