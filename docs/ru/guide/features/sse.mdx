## Что такое SSE

Server-Sent Events (SSE) — это веб-технология, позволяющая серверу отправлять данные клиенту. В отличие от WebSocket, SSE обеспечивает одностороннюю коммуникацию: данные передаются только от сервера клиенту. Это идеально подходит для сценариев, где требуется реальное время обновления данных с сервера без необходимости частой отправки данных от клиента. SSE работает поверх HTTP-протокола, легко реализуется и имеет встроенный механизм переподключения при разрыве соединения.

Предоставляет middleware с поддержкой `SSE`.

## Интеграция с большими языковыми моделями

При использовании OpenAI SDK или других SDK для работы с большими языковыми моделями потоковая передача результатов (Stream) является распространённым сценарием. SSE предоставляет идеальное решение, позволяя в реальном времени доставлять сгенерированный ИИ контент на фронтенд, что значительно улучшает пользовательский опыт.

Например, при подключении Chat Completion API от OpenAI с потоковым ответом можно использовать функционал SSE в Salvo для постепенной отправки сгенерированного ИИ контента клиенту, избавляя пользователей от необходимости ожидания полного ответа.
