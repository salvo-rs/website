# ID de Traza de Solicitud

import { Tab, Tabs } from '@rspress/core/theme';

El middleware de ID de solicitud es bastante flexible. El generador (IdGenerator) se utiliza para crear IDs, y puedes definir tu propio generador siempre que implementes el trait `IdGenerator`. El generador predeterminado es `UlidGenerator`.

Además, puedes controlar si se sobrescribe un `requestid` ya existente. También es posible configurar `header_name`, entre otros ajustes. Para más detalles, consulta la [documentación](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
