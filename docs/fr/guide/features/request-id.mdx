# ID de chaîne de requête

import { Tab, Tabs } from '@rspress/core/theme';

Le middleware Request Id est relativement flexible. Le générateur (IdGenerator) est utilisé pour générer l'ID, et vous pouvez définir votre propre générateur d'ID tant qu'il implémente le trait `IdGenerator`. Le générateur par défaut fourni est `UlidGenerator`.

De plus, vous pouvez contrôler s'il faut écraser l'`requestid` déjà existant. Vous pouvez également définir `header_name`, entre autres. Pour plus de détails, veuillez [consulter la documentation](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
