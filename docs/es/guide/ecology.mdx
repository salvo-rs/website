# Ecosistema de Salvo

Salvo cuenta con un ecosistema rico que incluye módulos mantenidos por la comunidad, proyectos construidos sobre Salvo y abundantes recursos tutoriales. Estos recursos pueden ayudarte a utilizar mejor el framework Salvo para construir aplicaciones web de alto rendimiento.
Además, en los subdirectorios se comparten algunas bibliotecas que funcionan bien con Salvo. El lenguaje Rust está en rápido desarrollo y sus bibliotecas se caracterizan por ser pequeñas y numerosas. Esperamos que estas bibliotecas comunes te ayuden a familiarizarte más rápido con el ecosistema de Rust, convirtiéndose en el bolsillo de tu Doraemon.
