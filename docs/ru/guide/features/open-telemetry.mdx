# Интеграция с OpenTelemetry

OpenTelemetry — это фреймворк с открытым исходным кодом для обеспечения наблюдаемости, предоставляющий стандартизированные способы сбора и экспорта телеметрических данных приложений (включая трассировку, метрики и логи). Он помогает разработчикам отслеживать производительность приложений, устранять проблемы и понимать поведение распределённых систем.

Основные преимущества OpenTelemetry:
- Унифицированный API и инструменты без привязки к вендору
- Поддержка множества языков программирования и бэкенд-систем
- Низкие накладные расходы при сборе данных
- Мощная поддержка распределённой трассировки

Предоставляется middleware с поддержкой OpenTelemetry. Пример можно посмотреть в [официальном репозитории](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger).
