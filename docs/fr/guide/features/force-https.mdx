# Forcer HTTPS

Le middleware `force-https` permet de rediriger toutes les requêtes vers le protocole HTTPS.

Si ce middleware est appliqué à un `Router`, la redirection de protocole ne se déclenche que lorsque la route correspond. Si la page n'existe pas, aucune redirection n'est effectuée.

Cependant, il est plus courant de vouloir rediriger automatiquement toutes les requêtes, même en cas d'erreur `404` lorsque la route ne correspond pas. Dans ce cas, le middleware peut être ajouté au niveau du `Service`. Les middlewares ajoutés au `Service` s'exécutent toujours, que la requête corresponde ou non à une route.
