## Introducción a WebSocket

WebSocket es un protocolo de comunicación full-duplex sobre una única conexión TCP. Facilita el intercambio de datos entre cliente y servidor, permitiendo que el servidor envíe datos activamente al cliente. En el protocolo WebSocket, el navegador y el servidor solo necesitan realizar un apretón de manos para establecer una conexión persistente y realizar transmisión bidireccional de datos.

A diferencia de las solicitudes HTTP tradicionales, la conexión WebSocket permanece abierta después de establecerse, evitando conexiones y desconexiones frecuentes, lo que reduce significativamente la sobrecarga de comunicación. Esto hace que WebSocket sea especialmente adecuado para aplicaciones en tiempo real, como juegos en línea, aplicaciones de chat, cotizaciones de acciones y otras aplicaciones que requieren actualizaciones en tiempo real.

Salvo proporciona una API simple pero poderosa a través de su middleware WebSocket, permitiendo a los desarrolladores integrar fácilmente la funcionalidad WebSocket en sus aplicaciones.
