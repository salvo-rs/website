## Introdução ao WebSocket

WebSocket é um protocolo de comunicação full-duplex sobre uma única conexão TCP. Ele simplifica a troca de dados entre cliente e servidor, permitindo que o servidor envie dados ativamente para o cliente. No protocolo WebSocket, o navegador e o servidor só precisam realizar um handshake uma vez para estabelecer uma conexão persistente e realizar transmissão bidirecional de dados.

Diferente das requisições HTTP tradicionais, a conexão WebSocket permanece aberta após ser estabelecida, evitando conexões e desconexões frequentes, o que reduz significativamente a sobrecarga de comunicação. Isso torna o WebSocket especialmente adequado para cenários de aplicações em tempo real, como jogos online, aplicativos de chat, cotações de ações e outros aplicativos que exigem atualizações instantâneas.

O Salvo fornece uma API concisa e poderosa por meio de seu middleware WebSocket, permitindo que os desenvolvedores integrem facilmente a funcionalidade WebSocket em suas aplicações.
