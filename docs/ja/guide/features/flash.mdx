# Flash

フラッシュメッセージ機能を提供するミドルウェア。

`FlashStore`はデータの読み書き操作を提供します。`CookieStore`はデータを`Cookie`に保存し、`SessionStore`はデータを`Session`に保存します。`SessionStore`は`session`機能と併用する必要があります。
