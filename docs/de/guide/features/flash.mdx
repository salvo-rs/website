# Flash

Middleware zur Bereitstellung von Flash-Nachrichten-Funktionalität.

`FlashStore` ermöglicht Lese- und Schreibzugriffe auf Daten. `CookieStore` speichert Daten in `Cookies`. Während `SessionStore` Daten in der `Session` speichert, muss `SessionStore` zusammen mit der `session`-Funktionalität verwendet werden.
