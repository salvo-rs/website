# Integración con OpenTelemetry

OpenTelemetry es un marco de observabilidad de código abierto que proporciona métodos estandarizados para recopilar y exportar datos de telemetría de aplicaciones (incluyendo rastreos, métricas y registros). Ayuda a los desarrolladores a monitorear el rendimiento de las aplicaciones, solucionar problemas y comprender el comportamiento de los sistemas distribuidos.

Principales ventajas de OpenTelemetry:
- API y herramientas unificadas, sin dependencia de proveedores
- Soporte para múltiples lenguajes de programación y sistemas backend
- Recopilación de datos con baja sobrecarga
- Potente soporte para rastreo distribuido

Proporcionamos un middleware con soporte para OpenTelemetry. Puedes consultar el [ejemplo](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) en el repositorio oficial.
