El middleware Request Id es bastante flexible. El generador (IdGenerator) se utiliza para crear IDs, y puedes definir tu propio generador de IDs siempre que implementes el trait `IdGenerator`. El generador predeterminado proporcionado es `UlidGenerator`.

Además, puedes controlar si deseas sobrescribir un `requestid` ya existente. También puedes configurar `header_name`, entre otros. Para más detalles, consulta la [documentación](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
