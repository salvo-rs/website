# Intégration d'OpenTelemetry

OpenTelemetry est un cadre d'observabilité open source qui offre une méthode standardisée pour collecter et exporter les données de télémétrie des applications (y compris les traces, les métriques et les journaux). Il aide les développeurs à surveiller les performances des applications, à résoudre les problèmes et à comprendre le comportement des systèmes distribués.

Principaux avantages d'OpenTelemetry :
- API et outils unifiés, sans verrouillage de fournisseur
- Prise en charge de plusieurs langages de programmation et systèmes backend
- Collecte de données à faible surcharge
- Prise en charge robuste du traçage distribué

Fournit un middleware prenant en charge OpenTelemetry. Vous pouvez consulter l'[exemple](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) dans la bibliothèque officielle.
