# Middleware de Límite de Tamaño para Carga de Archivos

import { Tab, Tabs } from '@rspress/core/theme';

Un middleware para limitar el tamaño de los archivos cargados en las solicitudes.

## Casos de Uso

El Limitador de Tamaño se utiliza principalmente en los siguientes escenarios:
- Evitar que los usuarios carguen archivos excesivamente grandes que consuman recursos del servidor
- Prevenir el desbordamiento de memoria del servidor causado por atacantes maliciosos que carguen grandes cantidades de datos
- Establecer políticas de límite de tamaño de solicitud diferentes para distintos endpoints de API
- Mejorar la eficiencia y estabilidad del procesamiento de solicitudes del servidor
