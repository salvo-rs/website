# 日誌中介軟體

日誌記錄在Web應用程式中扮演著至關重要的角色，它能夠：
- 提供請求處理過程的詳細資訊，幫助開發者追蹤應用行為
- 輔助問題排查與除錯，特別在生產環境中
- 監控應用效能和資源使用情況
- 記錄使用者存取模式和系統異常
- 滿足安全稽核和合規性要求

Salvo提供基本的Log功能中介軟體。若中介軟體直接加到`Router`上，將無法捕獲所有`Router`都不匹配時返回的404錯誤，建議添加到`Service`上。
