# 強制 HTTPS

`force-https` 中介軟體可以強制所有的請求轉向至使用 HTTPS 協定。

此中介軟體若應用於 `Router`，則只有在路由匹配成功時才會強制轉換協定，若遇到頁面不存在的情況，則不會進行轉向。

但更常見的需求是期望任何請求都能自動轉向，即使在路由未能匹配、返回 `404` 錯誤時。這時可以將中介軟體添加到 `Service` 上。無論請求是否被路由成功匹配，`Service` 上添加的中介軟體總是會執行。
