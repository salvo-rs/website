## Introdução ao WebSocket

WebSocket é um protocolo que permite comunicação full-duplex sobre uma única conexão TCP. Ele simplifica a troca de dados entre clientes e servidores, permitindo que o servidor envie dados ativamente para o cliente. No protocolo WebSocket, o navegador e o servidor só precisam realizar um handshake uma vez para estabelecer uma conexão persistente e habilitar a transmissão bidirecional de dados.

Diferente das requisições HTTP tradicionais, uma conexão WebSocket permanece aberta após ser estabelecida, evitando a configuração e encerramento frequentes de conexões, o que reduz significativamente a sobrecarga de comunicação. Isso torna o WebSocket especialmente adequado para aplicações em tempo real, como jogos online, aplicativos de chat, cotação de ações e outras aplicações que exigem atualizações em tempo real.

O Salvo fornece uma API concisa e poderosa através do seu middleware WebSocket, permitindo que os desenvolvedores integrem facilmente a funcionalidade WebSocket em suas aplicações.
