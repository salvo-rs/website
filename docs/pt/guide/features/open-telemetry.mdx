# Integração OpenTelemetry

O OpenTelemetry é uma estrutura de observabilidade de código aberto que oferece uma abordagem padronizada para coletar e exportar dados de telemetria de aplicativos (incluindo rastreamento, métricas e logs). Ele auxilia desenvolvedores a monitorar o desempenho de aplicações, solucionar problemas e compreender o comportamento de sistemas distribuídos.

Principais vantagens do OpenTelemetry:
- API e ferramentas unificadas, sem dependência de fornecedores
- Suporte a múltiplas linguagens de programação e sistemas de backend
- Coleta de dados com baixa sobrecarga
- Suporte robusto para rastreamento distribuído

Oferece um middleware com suporte ao OpenTelemetry. Consulte o [exemplo](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) no repositório oficial.
