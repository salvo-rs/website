# Écosystème Salvo

Salvo dispose d'un écosystème riche, comprenant des modules maintenus par la communauté, des projets construits sur Salvo et des ressources pédagogiques variées. Ces ressources vous aideront à mieux utiliser le framework Salvo pour développer des applications Web hautes performances.
Par ailleurs, les sous-répertoires partagent certaines bibliothèques qui fonctionnent bien avec Salvo. Le langage Rust est en plein essor, avec des bibliothèques souvent petites mais nombreuses. Nous espérons que ces bibliothèques courantes vous aideront à vous familiariser rapidement avec l'écosystème Rust, comme une poche magique de Doraemon.
