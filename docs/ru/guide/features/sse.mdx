## Что такое SSE

Server-Sent Events (SSE) — это веб-технология, позволяющая серверу отправлять данные клиенту. В отличие от WebSocket, SSE обеспечивает одностороннюю связь: данные передаются только от сервера клиенту. Этот подход идеально подходит для сценариев, где требуется реальное время отправки данных с сервера, но клиенту не нужно часто отправлять данные обратно. SSE работает на основе протокола HTTP, отличается простотой реализации и имеет встроенный механизм повторного подключения при разрыве соединения.

Предоставляет промежуточное ПО с поддержкой `SSE`.

## Интеграция с большими языковыми моделями

При подключении AI через OpenAI SDK или другие SDK больших языковых моделей часто возникает необходимость в потоковой передаче результатов (Stream). SSE предлагает идеальное решение, позволяя в реальном времени отправлять сгенерированный AI контент на фронтенд, что значительно улучшает пользовательский опыт.

Например, при использовании Chat Completion API от OpenAI с потоковыми ответами можно задействовать функционал SSE в Salvo для постепенной отправки сгенерированного AI контента клиенту, избавляя пользователей от необходимости ожидания полного ответа и создавая ощущение минимальной задержки.
