# Middleware de journalisation

La journalisation joue un rôle essentiel dans les applications web, permettant de :
- Fournir des informations détaillées sur le traitement des requêtes, aidant les développeurs à suivre le comportement de l'application
- Faciliter le dépannage et le débogage, particulièrement en environnement de production
- Surveiller les performances de l'application et l'utilisation des ressources
- Enregistrer les modèles d'accès des utilisateurs et les anomalies système
- Satisfaire aux exigences d'audit de sécurité et de conformité

Salvo fournit un middleware avec des fonctionnalités de journalisation de base. Si le middleware est ajouté directement au `Router`, il ne pourra pas capturer les erreurs 404 renvoyées lorsque aucune correspondance n'est trouvée dans le `Router`. Il est recommandé de l'ajouter au `Service`.
