# Flash

Middleware que proporciona funcionalidad de mensajes flash.

`FlashStore` ofrece operaciones de lectura y escritura de datos. `CookieStore` almacena los datos en `Cookies`, mientras que `SessionStore` los guarda en la `Sesión`. `SessionStore` debe utilizarse junto con la funcionalidad de `session`.
