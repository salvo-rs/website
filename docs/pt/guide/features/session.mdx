# Sessão

import { Tab, Tabs } from '@rspress/core/theme';

Middleware que fornece suporte para `Sessão`.

## O que é Sessão

Sessão refere-se aos dados do usuário armazenados no lado do servidor, normalmente usados para rastrear o estado do usuário e armazenar dados temporários. Em aplicações web, a Sessão aborda a natureza sem estado do protocolo HTTP, permitindo que o servidor reconheça múltiplas solicitações do mesmo usuário.

## Usando Sessão no Salvo

No Salvo, o middleware de Sessão oferece uma API simples para leitura e escrita de dados de sessão. O `SessionDepot` pode ser usado para obter uma instância de sessão, enquanto métodos como `get`, `set` e `remove` permitem gerenciar os dados da sessão.

## Casos de Uso Comuns

- Autenticação de usuário e gerenciamento de estado de login
- Armazenamento temporário de dados de formulário
- Armazenamento de configurações de preferência do usuário
