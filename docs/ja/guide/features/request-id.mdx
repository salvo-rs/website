# リクエストトレースID

import { Tab, Tabs } from '@rspress/core/theme';

Request Id ミドルウェアは柔軟性が高く、ID生成器(IdGenerator)はIDの生成に使用されます。独自のID生成器を定義することができ、`IdGenerator` トレイトを実装するだけで済みます。デフォルトで提供される生成器は `UlidGenerator` です。

さらに、既存の `requestid` を上書きするかどうかを制御できます。また、`header_name` などの設定も可能です。詳細については[ドキュメントを参照してください](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html)。
