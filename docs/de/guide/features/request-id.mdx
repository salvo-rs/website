# Request-ID

import { Tab, Tabs } from '@rspress/core/theme';

Die Request-ID-Middleware ist flexibel einsetzbar. Der Generator (IdGenerator) dient zur Erzeugung von IDs – Sie können Ihren eigenen ID-Generator definieren, solange das Trait `IdGenerator` implementiert wird. Standardmäßig wird der `UlidGenerator` bereitgestellt.

Zudem können Sie steuern, ob bereits vorhandene `requestid` überschrieben werden sollen. Einstellungen wie `header_name` sind ebenfalls möglich. Details finden Sie in der [Dokumentation](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
