# Flash

提供 Flash Message 的功能的中间件.

`FlashStore` 提供对数据的存取操作. `CookieStore` 会在 `Cookie` 中存储数据. 而 `SessionStore` 把数据存储在 `Session` 中, `SessionStore` 必须和 `session` 功能一起使用.
