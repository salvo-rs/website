# Integrazione OpenTelemetry

OpenTelemetry è un framework open source per l'osservabilità che fornisce metodi standardizzati per raccogliere ed esportare dati di telemetria dalle applicazioni (inclusi tracce, metriche e log). Aiuta gli sviluppatori a monitorare le prestazioni delle applicazioni, risolvere problemi e comprendere il comportamento dei sistemi distribuiti.

Vantaggi principali di OpenTelemetry:
- API e strumenti unificati, senza vincoli di fornitori
- Supporto per multipli linguaggi di programmazione e sistemi backend
- Raccolta dati a basso overhead
- Solido supporto per il tracciamento distribuito

Forniamo un middleware con supporto OpenTelemetry. È possibile fare riferimento all'[esempio](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) nella libreria ufficiale.
