# Flash

Middleware fournissant la fonctionnalité de messages flash.

`FlashStore` permet les opérations de lecture et d'écriture des données. `CookieStore` stocke les données dans les `Cookies`. Tandis que `SessionStore` conserve les données dans la `Session` et doit être utilisé conjointement avec la fonctionnalité `session`.
