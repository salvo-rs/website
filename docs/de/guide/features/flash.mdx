# Flash

Ein Middleware, der Flash-Message-Funktionalität bereitstellt.

`FlashStore` bietet Daten-Speicher- und Abrufoperationen. `CookieStore` speichert Daten in `Cookies`, während `SessionStore` Daten in `Session` speichert. `SessionStore` muss in Verbindung mit dem `session`-Feature verwendet werden.
