# HTTPS強制転送

`force-https`ミドルウェアは、すべてのリクエストをHTTPSプロトコル使用へ強制的に転送します。

このミドルウェアを`Router`に適用した場合、ルートが一致した時のみプロトコル転送が強制されます。ページが存在しない場合は転送されません。

しかしより一般的なニーズは、ルートが一致せず`404`エラーが返される場合も含め、あらゆるリクエストを自動転送することです。その場合はミドルウェアを`Service`に追加します。`Service`に追加されたミドルウェアは、リクエストがルートに正常にマッチしたかどうかに関係なく、常に実行されます。
