## O que é SSE

Eventos Enviados pelo Servidor (SSE) é uma tecnologia web que permite que servidores enviem dados para clientes. Diferente do WebSocket, o SSE é um método de comunicação unidirecional, onde os dados só podem ser enviados do servidor para o cliente. É adequado para cenários em que o servidor precisa enviar dados em tempo real, mas o cliente não precisa enviar dados com frequência. O SSE é baseado no protocolo HTTP, fácil de implementar e possui mecanismos de reconexão integrados.

Fornece suporte de middleware para `SSE`.

## Integração com Modelos de Linguagem de Grande Porte

Ao integrar IA usando o SDK da OpenAI ou outros SDKs de modelos de linguagem de grande porte, respostas em fluxo contínuo (Stream) são um requisito comum. O SSE fornece uma solução perfeita, permitindo a entrega em tempo real do conteúdo gerado por IA para o frontend, proporcionando assim uma melhor experiência do usuário.

Por exemplo, ao usar a API de Conclusão de Chat da OpenAI com respostas em fluxo contínuo, a funcionalidade SSE do Salvo pode ser aproveitada para enviar o conteúdo gerado por IA de forma incremental para o cliente, eliminando o atraso de espera por uma resposta completa.
