Request Id 中介軟體具有高度靈活性，生成器(IdGenerator) 用於產生 ID，您可以定義自己的 ID 生成器，只需實現 `IdGenerator` 這個 trait。預設提供的生成器是 `UlidGenerator`。

此外，您可以控制是否覆蓋現有已存在的 `requestid`。還能設置 `header_name` 等參數，詳情請[查閱文件](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html)。
