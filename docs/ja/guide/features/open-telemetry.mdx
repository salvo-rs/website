# OpenTelemetry連携

OpenTelemetryはオープンソースのオブザーバビリティフレームワークで、アプリケーションのテレメトリーデータ（トレース、メトリクス、ログを含む）を収集・エクスポートする標準化された方法を提供します。開発者がアプリケーションのパフォーマンスを監視し、問題を調査し、分散システムの動作を理解するのに役立ちます。

OpenTelemetryの主な利点:
- ベンダーロックインのない統一APIとツール
- 複数のプログラミング言語とバックエンドシステムのサポート
- 低オーバーヘッドのデータ収集
- 分散トレーシングへの強力なサポート

OpenTelemetryをサポートするミドルウェアを提供しています。公式ライブラリの[サンプル](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger)を参照してください。
