## WebSocket 概要

WebSocket は、単一の TCP 接続上で全二重通信を行うプロトコルです。WebSocket により、クライアントとサーバー間のデータ交換がより簡素化され、サーバーからクライアントへの能動的なデータプッシュが可能になります。WebSocket プロトコルでは、ブラウザとサーバーは一度のハンドシェイクを完了するだけで、持続的な接続を確立し、双方向のデータ転送を行うことができます。

従来の HTTP リクエストとは異なり、WebSocket 接続は確立後に開いたまま維持されるため、頻繁な接続の確立と切断を回避し、通信オーバーヘッドを大幅に削減します。これにより、WebSocket はオンラインゲーム、チャットアプリケーション、株価情報など、リアルタイム更新が必要なアプリケーションに特に適しています。

Salvo は WebSocket ミドルウェアを通じて、簡潔で強力な API を提供し、開発者がアプリケーションに簡単に WebSocket 機能を統合できるようにします。
