# Intégration d'OpenTelemetry

OpenTelemetry est un cadre d'observabilité open source qui fournit des méthodes standardisées pour collecter et exporter les données de télémétrie des applications (y compris les traces, les métriques et les journaux). Il aide les développeurs à surveiller les performances des applications, à résoudre les problèmes et à comprendre le comportement des systèmes distribués.

Principaux avantages d'OpenTelemetry :
- API et outils unifiés, sans dépendance à un fournisseur spécifique
- Prise en charge de multiples langages de programmation et systèmes backend
- Collecte de données à faible impact
- Excellente prise en charge du traçage distribué

Middleware fournissant la prise en charge d'OpenTelemetry. Vous pouvez vous référer à l'[exemple](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) dans la bibliothèque officielle.
