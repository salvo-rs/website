# ID Catena Richieste

import { Tab, Tabs } from '@rspress/core/theme';

Il middleware Request ID è altamente flessibile. Il generatore di ID (IdGenerator) viene utilizzato per generare gli identificatori, ed è possibile definire un proprio generatore purché implementi il tratto `IdGenerator`. Il generatore predefinito fornito è `UlidGenerator`.

Inoltre, è possibile controllare se sovrascrivere un `requestid` esistente. È anche possibile impostare `header_name` e altre configurazioni. Per i dettagli, consultare [la documentazione](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
