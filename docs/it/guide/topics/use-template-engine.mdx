# Utilizzo dei motori di template

import { Tab, Tabs } from '@rspress/core/theme';

Salvo non include alcun motore di template integrato, dopotutto, la preferenza per uno stile di motore di template varia da persona a persona.

In sostanza, un motore di template è: dati + template = stringa.

Pertanto, qualsiasi motore di template può essere supportato purché sia in grado di generare la stringa finale.

Ad esempio, il supporto per `askama`:

**Nota**: Per progetti non particolarmente complessi, consigliamo un approccio di sviluppo con separazione tra frontend e backend, utilizzando framework UI più flessibili e con un ecosistema più ricco (come React, Vue, Svelte, ecc.) per costruire il frontend, mentre Salvo funge da servizio API backend. Questo metodo offre una maggiore efficienza di sviluppo, responsabilità più chiare tra frontend e backend, ed è più in linea con le tendenze di sviluppo delle applicazioni web moderne.
