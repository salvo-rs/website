# Ограничение размера загружаемых файлов
Промежуточное ПО

import { Tab, Tabs } from '@rspress/core/theme';

Предоставляет промежуточное ПО для ограничения размера загружаемых файлов в запросах.

## Сценарии применения

Size Limiter в основном используется в следующих случаях:
- Предотвращение загрузки пользователями слишком больших файлов, которые могут занять ресурсы сервера
- Избежание атак злоумышленников, которые могут привести к переполнению памяти сервера путем загрузки большого объема данных
- Установка различных стратегий ограничения размера запросов для различных конечных точек API
- Повышение эффективности и стабильности обработки запросов сервером
