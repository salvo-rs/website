# Flash

Middleware fournissant la fonctionnalité de messages Flash.

`FlashStore` permet les opérations de lecture et d'écriture des données. `CookieStore` stocke les données dans les `Cookies`. Quant à `SessionStore`, il conserve les données dans la `Session` et doit être utilisé conjointement avec la fonctionnalité `session`.
