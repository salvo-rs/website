## Cosa sono gli SSE

Gli Server-Sent Events (SSE) sono una tecnologia web che consente al server di inviare dati al client in modalità push. A differenza di WebSocket, gli SSE sono una comunicazione unidirezionale: i dati possono essere inviati solo dal server al client, il che li rende ideali per scenari in cui il server deve inviare dati in tempo reale senza che il client debba trasmettere frequentemente. Basati sul protocollo HTTP, gli SSE sono semplici da implementare e includono meccanismi automatici di riconnessione in caso di interruzioni.

## Integrazione con modelli linguistici di grandi dimensioni

Quando si integra l'IA utilizzando SDK come OpenAI o altri SDK per modelli linguistici di grandi dimensioni, la restituzione dei risultati in modalità streaming (Stream) è uno scenario comune. Gli SSE forniscono una soluzione perfetta, consentendo di inviare in tempo reale i contenuti generati dall'IA al frontend, offrendo un'esperienza utente migliore.

Ad esempio, quando si utilizza l'API Chat Completion di OpenAI con risposte in streaming, è possibile sfruttare la funzionalità SSE di Salvo per inviare gradualmente i contenuti generati dall'IA al client, evitando che l'utente debba attendere il completamento della risposta.
