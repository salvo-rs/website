Middleware that provides size limits for uploaded files in requests.

## Use Cases

Size Limiter is primarily used in the following scenarios:
- Preventing users from uploading files that are too large and consuming server resources
- Avoiding malicious attacks where attackers upload large amounts of data causing server memory overflow
- Setting different request size limit policies for different API endpoints
- Improving the efficiency and stability of server request processing
