# Integración de OpenTelemetry

OpenTelemetry es un marco de observabilidad de código abierto que proporciona una forma estandarizada de recopilar y exportar datos de telemetría de aplicaciones (incluyendo seguimiento, métricas y registros). Ayuda a los desarrolladores a monitorear el rendimiento de las aplicaciones, solucionar problemas y comprender el comportamiento de los sistemas distribuidos.

Principales ventajas de OpenTelemetry:
- API y herramientas unificadas, sin dependencia de proveedores
- Compatibilidad con múltiples lenguajes de programación y sistemas backend
- Bajo impacto en la recopilación de datos
- Sólido soporte para seguimiento distribuido

Proporciona un middleware con soporte para OpenTelemetry. Puedes consultar el [ejemplo](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) en la biblioteca oficial.
