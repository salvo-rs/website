# 會話

提供支援 `Session` 功能的中介軟體。

## 什麼是會話

會話是伺服器端儲存的一段使用者資料，通常用於追蹤使用者狀態和暫存臨時資料。在網頁應用中，會話能夠解決 HTTP 協定無狀態的問題，讓伺服器能夠識別來自同一使用者的多次請求。

## Salvo 中的會話使用

在 Salvo 中，會話中介軟體提供了簡潔的 API 來讀取和寫入會話資料。透過 `SessionDepot` 可以取得會話實例，並使用 `get`、`set`、`remove` 等方法來管理會話資料。

## 常見用途

- 使用者身份驗證與登入狀態管理
- 表單資料暫存
- 使用者偏好設定儲存
