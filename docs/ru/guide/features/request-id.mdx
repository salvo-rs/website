# Идентификатор цепочки запросов

import { Tab, Tabs } from '@rspress/core/theme';

Промежуточное ПО для идентификации запросов обладает высокой гибкостью. Генератор идентификаторов (IdGenerator) используется для создания ID, и вы можете определить собственный генератор, при условии что он реализует трейт `IdGenerator`. По умолчанию предоставляется генератор `UlidGenerator`.

Кроме того, вы можете контролировать, следует ли перезаписывать существующий `requestid`. Также можно настроить `header_name` и другие параметры. Подробности смотрите в [документации](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
