# 請求鏈路識別碼

import { Tab, Tabs } from '@rspress/core/theme';

Request Id 中介軟體具備高度彈性，其生成器(IdGenerator) 專門用於產生識別碼，您可以自訂專屬的識別碼生成器，只需實作 `IdGenerator` 這個特徵。系統預設提供的生成器為 `UlidGenerator`。

此外，您可自行控制是否覆蓋現有已存在的 `requestid`。還能設定 `header_name` 等參數，詳細內容請參閱[官方文件](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html)。
