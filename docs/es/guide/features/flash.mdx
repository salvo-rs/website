# Flash

Middleware que proporciona funcionalidad de mensajes flash.

`FlashStore` ofrece operaciones de lectura y escritura de datos. `CookieStore` almacena los datos en `Cookies`, mientras que `SessionStore` guarda la información en la `Sesión`. El uso de `SessionStore` requiere la funcionalidad de `session`.
