## Introduzione a WebSocket

WebSocket è un protocollo di comunicazione full-duplex su una singola connessione TCP. WebSocket semplifica lo scambio di dati tra client e server, consentendo al server di inviare attivamente dati al client. Nel protocollo WebSocket, browser e server devono completare solo una stretta di mano per stabilire una connessione persistente e trasmettere dati bidirezionalmente.

A differenza delle tradizionali richieste HTTP, la connessione WebSocket rimane aperta dopo essere stata stabilita, evitando frequenti connessioni e disconnessioni e riducendo notevolmente il sovraccarico di comunicazione. Ciò rende WebSocket particolarmente adatto per scenari applicativi in tempo reale, come giochi online, applicazioni di chat, quotazioni azionarie e altre applicazioni che richiedono aggiornamenti in tempo reale.

Salvo fornisce un'API concisa e potente attraverso il suo middleware WebSocket, consentendo agli sviluppatori di integrare facilmente le funzionalità WebSocket nelle loro applicazioni.
