## Introduction à WebSocket

WebSocket est un protocole permettant une communication full-duplex sur une seule connexion TCP. Il simplifie l'échange de données entre client et serveur en autorisant le serveur à envoyer activement des données au client. Avec le protocole WebSocket, le navigateur et le serveur n'ont besoin que d'une seule poignée de main pour établir une connexion persistante permettant des transferts de données bidirectionnels.

Contrairement aux requêtes HTTP traditionnelles, une connexion WebSocket reste ouverte une fois établie, évitant ainsi des connexions et déconnexions fréquentes et réduisant considérablement la surcharge de communication. Cela rend WebSocket particulièrement adapté aux applications en temps réel comme les jeux en ligne, les applications de chat, les cotations boursières et autres programmes nécessitant des mises à jour instantanées.

Salvo propose via son middleware WebSocket une API simple mais puissante permettant aux développeurs d'intégrer facilement des fonctionnalités WebSocket dans leurs applications.
