# Сжатие ответов

Промежуточное ПО для сжатия содержимого `Response`.

Поддерживает три формата сжатия: `br`, `gzip` и `deflate`. Приоритет и конфигурация каждого метода сжатия могут быть настроены по необходимости.

## Сжатие данных в протоколе HTTP

1. Уровень формата файла: определённые типы файлов используют специализированные алгоритмы сжатия.
2. Уровень протокола HTTP: сквозная передача сжатых ресурсов осуществляется посредством кодирования содержимого.
3. Уровень соединения: сжатие данных выполняется между узлами HTTP-соединения.

Для получения дополнительной информации о сжатии в HTTP обратитесь к [документации Mozilla Developer Network](https://developer.mozilla.org/en-US/docs/Web/HTTP/Guides/Compression).
