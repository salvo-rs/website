# Integrazione OpenTelemetry

OpenTelemetry è un framework open-source di osservabilità che offre un approccio standardizzato per raccogliere ed esportare dati di telemetria dalle applicazioni, inclusi tracce, metriche e log. Aiuta gli sviluppatori a monitorare le prestazioni delle applicazioni, risolvere problemi e comprendere il comportamento dei sistemi distribuiti.

Vantaggi principali di OpenTelemetry:
- API e strumenti unificati senza vincoli verso fornitori specifici
- Supporto per più linguaggi di programmazione e sistemi backend
- Raccolta dati a basso impatto
- Solido supporto per la tracciatura distribuita

Offre middleware con supporto OpenTelemetry. Puoi fare riferimento all'[esempio](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) nel repository ufficiale.
