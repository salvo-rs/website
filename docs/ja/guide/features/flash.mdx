# フラッシュ

フラッシュメッセージ機能を提供するミドルウェアです。

`FlashStore` はデータの読み書き操作を提供します。`CookieStore` はデータを `Cookie` に保存し、`SessionStore` はデータを `Session` に保存します。`SessionStore` を使用するには `session` 機能と一緒に使用する必要があります。
