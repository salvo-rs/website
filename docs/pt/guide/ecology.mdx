# Ecossistema Salvo

O Salvo possui um ecossistema rico, incluindo módulos mantidos pela comunidade, projetos construídos com base no Salvo e uma variedade de recursos tutoriais. Esses recursos podem ajudá-lo a utilizar melhor o framework Salvo para construir aplicações web de alto desempenho.
Além disso, subdiretórios compartilham algumas bibliotecas que funcionam bem com o Salvo. A linguagem Rust está em rápido desenvolvimento e suas bibliotecas têm a característica de serem pequenas e numerosas. Esperamos que essas bibliotecas comuns possam ajudá-lo a se familiarizar mais rapidamente com o ecossistema Rust, tornando-se como o bolso do Doraemon.
