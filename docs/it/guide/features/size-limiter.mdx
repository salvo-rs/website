# Middleware per il Limite di Dimensione dei File Caricati

import { Tab, Tabs } from '@rspress/core/theme';

Fornisce un middleware per limitare la dimensione dei file caricati nelle richieste.

## Scenari di Applicazione

Size Limiter è principalmente utilizzato nei seguenti scenari:
- Impedire agli utenti di caricare file troppo grandi che consumano risorse del server
- Evitare che attaccanti malintenzionati causino overflow di memoria del server caricando grandi quantità di dati
- Impostare strategie diverse di limitazione della dimensione delle richieste per diversi endpoint API
- Migliorare l'efficienza e la stabilità del server nell'elaborazione delle richieste
