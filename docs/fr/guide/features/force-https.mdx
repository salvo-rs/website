# Forcer HTTPS

Le middleware `force-https` peut rediriger toutes les requêtes pour utiliser le protocole HTTPS.

Si ce middleware est appliqué à un `Router`, il n'appliquera la redirection de protocole que lorsqu'une route correspond. Si une page n'existe pas, aucune redirection ne se produira.

Cependant, une exigence plus courante est de rediriger automatiquement toute requête, même lorsque la route ne correspond pas et renvoie une erreur `404`. Dans de tels cas, le middleware peut être ajouté au `Service`. Les middlewares ajoutés au `Service` s'exécuteront toujours, que la requête soit correctement prise en charge par une route ou non.
