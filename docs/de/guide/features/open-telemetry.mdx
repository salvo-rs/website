# OpenTelemetry-Integration

OpenTelemetry ist ein Open-Source-Observability-Framework, das standardisierte Methoden zur Erfassung und Exportierung von Telemetriedaten (einschließlich Traces, Metriken und Logs) aus Anwendungen bereitstellt. Es unterstützt Entwickler bei der Überwachung der Anwendungsleistung, der Fehlerbehebung und dem Verständnis des Verhaltens verteilter Systeme.

Die Hauptvorteile von OpenTelemetry:
- Einheitliche APIs und Tools ohne Vendor-Lock-in
- Unterstützung für mehrere Programmiersprachen und Backend-Systeme
- Geringer Overhead bei der Datenerfassung
- Umfangreiche Unterstützung für verteilte Ablaufverfolgung

Bietet Middleware mit OpenTelemetry-Unterstützung. Ein [Beispiel](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) ist im offiziellen Repository verfügbar.
