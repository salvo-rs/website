Il middleware Request Id è piuttosto flessibile. Il generatore (IdGenerator) viene utilizzato per generare gli ID ed è possibile definire il proprio generatore di ID, purché implementi il trait `IdGenerator`. Il generatore predefinito fornito è `UlidGenerator`.

Inoltre, è possibile controllare se sovrascrivere un `requestid` già esistente. È anche possibile impostare `header_name` e altro. Per i dettagli, consultare la [documentazione](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
