# Сессия

Промежуточное ПО, предоставляющее поддержку `Session`.

## Что такое Session

Session (сессия) — это пользовательские данные, хранящиеся на стороне сервера, обычно используемые для отслеживания состояния пользователя и хранения временных данных. В веб-приложениях Session решает проблему отсутствия состояния у протокола HTTP, позволяя серверу распознавать несколько запросов от одного и того же пользователя.

## Использование Session в Salvo

В Salvo промежуточное ПО Session предоставляет простой API для чтения и записи данных сессии. `SessionDepot` можно использовать для получения экземпляра сессии, а методы `get`, `set` и `remove` позволяют управлять данными сессии.

## Типичные случаи использования

- Аутентификация пользователя и управление состоянием входа
- Временное хранение данных форм
- Хранение пользовательских настроек
