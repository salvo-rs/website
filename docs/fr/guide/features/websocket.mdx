## Introduction à WebSocket

WebSocket est un protocole de communication full-duplex sur une seule connexion TCP. Il simplifie l'échange de données entre le client et le serveur, permettant à ce dernier d'envoyer activement des données au client. Dans le protocole WebSocket, le navigateur et le serveur n'ont besoin d'effectuer qu'une seule poignée de main pour établir une connexion persistante et permettre une transmission de données bidirectionnelle.

Contrairement aux requêtes HTTP traditionnelles, une connexion WebSocket reste ouverte après son établissement, évitant ainsi des connexions et déconnexions fréquentes, ce qui réduit considérablement la surcharge de communication. Cela rend WebSocket particulièrement adapté aux applications en temps réel, telles que les jeux en ligne, les applications de chat, les cotations boursières et d'autres applications nécessitant des mises à jour en temps réel.

Salvo fournit une API simple et puissante via son middleware WebSocket, permettant aux développeurs d'intégrer facilement la fonctionnalité WebSocket dans leurs applications.
