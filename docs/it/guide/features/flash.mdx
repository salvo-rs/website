# Flash

Un middleware che fornisce funzionalità di Messaggi Flash.

`FlashStore` offre operazioni di archiviazione e recupero dati. `CookieStore` memorizza i dati nei `Cookie`, mentre `SessionStore` archivia i dati nella `Sessione`. `SessionStore` deve essere utilizzato in combinazione con la funzionalità `session`.
