Fornece um middleware para limitar o tamanho de arquivos enviados em requisições.

## Cenários de Aplicação

O Size Limiter é utilizado principalmente nos seguintes cenários:
- Prevenir que usuários enviem arquivos muito grandes que consumam recursos do servidor
- Evitar que atacantes maliciosos causem estouro de memória no servidor através do envio massivo de dados
- Definir políticas de limite de tamanho diferentes para diferentes endpoints de API
- Melhorar a eficiência e estabilidade do servidor no processamento de requisições
