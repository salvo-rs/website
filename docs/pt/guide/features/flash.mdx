# Flash

Middleware que fornece funcionalidade de Mensagens Flash.

`FlashStore` oferece operações de leitura e escrita de dados. `CookieStore` armazena dados em `Cookies`. Já `SessionStore` armazena dados na `Sessão`, e deve ser utilizado em conjunto com a funcionalidade de `session`.
