## Introduzione a WebSocket

WebSocket è un protocollo che consente la comunicazione full-duplex su una singola connessione TCP. Semplifica lo scambio di dati tra client e server permettendo al server di inviare attivamente dati al client. Nel protocollo WebSocket, il browser e il server devono eseguire solo una volta l'handshake per stabilire una connessione persistente e abilitare la trasmissione bidirezionale dei dati.

A differenza delle tradizionali richieste HTTP, una connessione WebSocket rimane aperta dopo essere stata stabilita, evitando frequenti configurazioni e chiusure di connessione, riducendo significativamente l'overhead di comunicazione. Ciò rende WebSocket particolarmente adatto per applicazioni in tempo reale, come giochi online, applicazioni di chat, ticker azionari e altre applicazioni che richiedono aggiornamenti in tempo reale.

Salvo fornisce un'API concisa e potente attraverso il suo middleware WebSocket, consentendo agli sviluppatori di integrare facilmente la funzionalità WebSocket nelle loro applicazioni.
