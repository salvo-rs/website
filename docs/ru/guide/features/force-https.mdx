# Принудительное HTTPS

Промежуточное ПО `force-https` может принудительно перенаправлять все запросы на использование протокола HTTPS.

Если этот промежуточный слой применяется к `Router`, перенаправление протокола будет выполняться только при совпадении маршрута. В случае отсутствия страницы перенаправление не произойдет.

Однако более распространенным требованием является автоматическое перенаправление любых запросов, даже если маршрут не совпадает и возвращается ошибка `404`. В этом случае промежуточное ПО можно добавить к `Service`. Промежуточное ПО, добавленное к `Service`, всегда выполняется, независимо от того, был ли запрос успешно сопоставлен с маршрутом.
