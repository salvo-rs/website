# Forzar HTTPS

El middleware `force-https` puede redirigir todas las solicitudes para que utilicen el protocolo HTTPS.

Si este middleware se aplica a un `Router`, solo aplicará la redirección de protocolo cuando se encuentre una ruta coincidente. Si una página no existe, no ocurrirá ninguna redirección.

Sin embargo, un requisito más común es redirigir automáticamente cualquier solicitud, incluso cuando la ruta no coincide y devuelve un error `404`. En tales casos, el middleware puede agregarse al `Service`. El middleware agregado al `Service` siempre se ejecutará, independientemente de si la solicitud coincide exitosamente con una ruta.
