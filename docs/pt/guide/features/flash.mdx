# Flash

Um middleware que fornece funcionalidade de Mensagens Flash.

`FlashStore` oferece operações de armazenamento e recuperação de dados. `CookieStore` armazena dados em `Cookies`, enquanto `SessionStore` armazena dados em `Sessão`. `SessionStore` deve ser usado em conjunto com o recurso `session`.
