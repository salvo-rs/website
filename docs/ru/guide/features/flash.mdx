# Flash

Промежуточное ПО, предоставляющее функциональность Flash-сообщений.

`FlashStore` обеспечивает операции хранения и извлечения данных. `CookieStore` сохраняет данные в `Cookies`, тогда как `SessionStore` хранит данные в `Session`. `SessionStore` должен использоваться совместно с функцией `session`.
