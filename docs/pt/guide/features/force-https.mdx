# Forçar HTTPS

O middleware `force-https` pode redirecionar todas as solicitações para usar o protocolo HTTPS.

Se este middleware for aplicado a um `Router`, a conversão de protocolo só ocorrerá quando a rota corresponder. Caso a página não exista, não haverá redirecionamento.

No entanto, é mais comum desejar que qualquer solicitação seja redirecionada automaticamente, mesmo quando a rota não corresponder e retornar um erro `404`. Nesse caso, o middleware pode ser adicionado ao `Service`. Independentemente de a solicitação corresponder com sucesso a uma rota, o middleware adicionado ao `Service` sempre será executado.
