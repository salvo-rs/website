# OpenTelemetry-Integration

OpenTelemetry ist ein Open-Source-Observability-Framework, das standardisierte Methoden zur Sammlung und Exportierung von Telemetriedaten (einschließlich Traces, Metriken und Logs) von Anwendungen bereitstellt. Es unterstützt Entwickler bei der Überwachung der Anwendungsleistung, der Fehlerbehebung und dem Verständnis des Verhaltens verteilter Systeme.

Die Hauptvorteile von OpenTelemetry:
- Vereinheitlichte APIs und Tools ohne Vendor-Lock-in
- Unterstützung für mehrere Programmiersprachen und Backend-Systeme
- Geringer Overhead bei der Datensammlung
- Umfassende Unterstützung für verteilte Ablaufverfolgung

Middleware mit OpenTelemetry-Unterstützung wird bereitgestellt. Ein [Beispiel](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) finden Sie im offiziellen Repository.
