# Integrazione OpenTelemetry

OpenTelemetry è un framework open source per l'osservabilità che fornisce un approccio standardizzato per raccogliere ed esportare dati di telemetria delle applicazioni (inclusi tracce, metriche e log). Aiuta gli sviluppatori a monitorare le prestazioni delle applicazioni, risolvere problemi e comprendere il comportamento dei sistemi distribuiti.

Vantaggi principali di OpenTelemetry:
- API e strumenti unificati, senza vincoli di fornitore
- Supporto per più linguaggi di programmazione e sistemi backend
- Raccolta dati a basso impatto
- Solido supporto per il tracciamento distribuito

Fornisce un middleware con supporto per OpenTelemetry. È possibile fare riferimento all'[esempio](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) nella libreria ufficiale.
