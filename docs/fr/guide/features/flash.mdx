# Flash

Un middleware qui fournit la fonctionnalité de messages Flash.

`FlashStore` propose des opérations de stockage et de récupération de données. `CookieStore` stocke les données dans les `Cookies`, tandis que `SessionStore` stocke les données dans la `Session`. `SessionStore` doit être utilisé conjointement avec la fonctionnalité `session`.
