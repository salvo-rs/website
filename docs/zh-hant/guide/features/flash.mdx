# 快閃訊息

提供快閃訊息功能的 middleware。

`FlashStore` 提供資料存取操作。`CookieStore` 會將資料儲存在 `Cookie` 中。而 `SessionStore` 則將資料儲存在 `Session` 裡，`SessionStore` 必須與 `session` 功能搭配使用。
