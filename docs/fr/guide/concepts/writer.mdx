# Writer

`Writer` est utilisé pour écrire du contenu dans `Response` :

```rust

pub trait Writer {
    async fn write(mut self, req: &mut Request, depot: &mut Depot, res: &mut Response);
}

pub trait Handler: Send + Sync + 'static {

```

Les différences principales sont :
- Leur utilité diffère : `Writer` représente l'écriture de contenu spécifique dans `Response`, implémentée par des contenus concrets tels que des chaînes de caractères, des messages d'erreur, etc. Alors que `Handler` est utilisé pour traiter l'intégralité de la requête.
- `Writer` est créé dans `Handler` et se consomme lors de l'appel de la fonction `write`, ce qui en fait un appel unique. En revanche, `Handler` est partagé par toutes les requêtes.
- `Writer` peut être utilisé comme contenu dans le `Result` retourné par `Handler`.
- `Writer` ne contient pas de paramètre `FlowCtrl` et ne peut donc pas contrôler le déroulement de l'exécution de la requête.
