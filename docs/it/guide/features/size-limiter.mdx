Fornisce un middleware per limitare la dimensione dei file caricati nelle richieste.

## Casi d'uso

Size Limiter è principalmente utilizzato nei seguenti scenari:
- Prevenire che gli utenti carichino file troppo grandi occupando risorse del server
- Evitare che attaccanti malintenzionati causino overflow di memoria sul server caricando grandi quantità di dati
- Impostare strategie diverse di limitazione per diversi endpoint API
- Migliorare l'efficienza e la stabilità del server nell'elaborazione delle richieste
