# Ecosistema Salvo

Salvo vanta un ricco ecosistema che include moduli mantenuti dalla comunità, progetti costruiti su Salvo e una vasta gamma di risorse didattiche. Queste risorse possono aiutarti a utilizzare meglio il framework Salvo per costruire applicazioni web ad alte prestazioni.
Inoltre, le sottodirectory condividono alcune librerie che si integrano bene con Salvo. Il linguaggio Rust è in rapida evoluzione e le sue librerie sono caratterizzate da dimensioni ridotte e numerose, speriamo che queste librerie comuni possano aiutarti a familiarizzare più rapidamente con l'ecosistema Rust, diventando come la tasca di Doraemon.
