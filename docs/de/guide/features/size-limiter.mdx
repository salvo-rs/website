Middleware zur Begrenzung der Größe hochgeladener Dateien in Anfragen.

## Anwendungsszenarien

Der Size Limiter wird hauptsächlich in folgenden Situationen eingesetzt:
- Verhindern, dass Benutzer zu große Dateien hochladen und Serverressourcen belegen
- Vermeiden von Server-Speicherüberläufen durch böswillige Angreifer, die große Datenmengen hochladen
- Festlegen unterschiedlicher Größenbeschränkungsstrategien für verschiedene API-Endpunkte
- Verbesserung der Effizienz und Stabilität der Serveranfrageverarbeitung
