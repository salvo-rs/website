# Интеграция OpenTelemetry

OpenTelemetry — это фреймворк с открытым исходным кодом для обеспечения наблюдаемости, предоставляющий стандартизированный способ сбора и экспорта телеметрических данных приложений (включая трассировку, метрики и логи). Он помогает разработчикам отслеживать производительность приложений, устранять проблемы и понимать поведение распределённых систем.

Основные преимущества OpenTelemetry:
- Унифицированный API и инструменты без привязки к конкретному поставщику
- Поддержка множества языков программирования и бэкенд-систем
- Низкие накладные расходы при сборе данных
- Мощная поддержка распределённой трассировки

Предоставляет промежуточное ПО с поддержкой OpenTelemetry. Пример можно найти в [официальном репозитории](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger).
