# Logging-Middleware

Die Protokollierung spielt eine entscheidende Rolle in Webanwendungen, da sie:

- Detaillierte Informationen zum Anfrageverarbeitungsprozess bereitstellt und Entwicklern hilft, das Anwendungsverhalten nachzuvollziehen
- Bei der Fehlerbehebung und Fehlersuche unterstützt, insbesondere in Produktionsumgebungen
- Die Anwendungsleistung und Ressourcennutzung überwacht
- Benutzerzugriffsmuster und Systemausnahmen erfasst
- Sicherheitsaudits und Compliance-Anforderungen erfüllt

Salvo bietet eine grundlegende Logging-Middleware. Wenn die Middleware direkt zum `Router` hinzugefügt wird, kann sie 404-Fehler nicht erfassen, die zurückgegeben werden, wenn keine Route übereinstimmt. Es wird empfohlen, sie stattdessen zum `Service` hinzuzufügen.
