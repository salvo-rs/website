O middleware Request Id é bastante flexível. O gerador (IdGenerator) é usado para criar IDs, e você pode definir seu próprio gerador de IDs, desde que implemente o trait `IdGenerator`. O gerador padrão fornecido é o `UlidGenerator`.

Além disso, você pode controlar se deseja substituir um `requestid` já existente. Também é possível configurar o `header_name`, entre outros. Para mais detalhes, [consulte a documentação](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
