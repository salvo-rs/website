# Integração OpenTelemetry

O OpenTelemetry é um framework de observabilidade de código aberto que fornece métodos padronizados para coletar e exportar dados de telemetria de aplicações (incluindo rastreamentos, métricas e logs). Ele auxilia desenvolvedores a monitorar o desempenho de aplicativos, solucionar problemas e compreender o comportamento de sistemas distribuídos.

Principais vantagens do OpenTelemetry:
- API e ferramentas unificadas, sem lock-in de fornecedor
- Suporte a múltiplas linguagens de programação e sistemas de backend
- Coleta de dados com baixo overhead
- Forte suporte para rastreamento distribuído

Fornecemos um middleware com suporte a OpenTelemetry. Você pode consultar o [exemplo](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) no repositório oficial.
