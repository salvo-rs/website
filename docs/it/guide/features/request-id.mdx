# ID Tracciamento Richieste

import { Tab, Tabs } from '@rspress/core/theme';

Il middleware Request Id è altamente flessibile: il generatore (IdGenerator) viene utilizzato per creare ID ed è possibile definire un proprio generatore implementando il trait `IdGenerator`. Il generatore predefinito fornito è `UlidGenerator`.

Inoltre, è possibile controllare se sovrascrivere l'`requestid` già esistente. È anche possibile impostare `header_name` e altre opzioni. Per i dettagli, consultare la [documentazione](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
