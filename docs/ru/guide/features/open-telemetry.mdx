# Интеграция OpenTelemetry

OpenTelemetry — это фреймворк с открытым исходным кодом для обеспечения наблюдаемости, который предоставляет стандартизированный подход к сбору и экспорту телеметрических данных из приложений, включая трассировки, метрики и логи. Он помогает разработчикам отслеживать производительность приложений, устранять неполадки и понимать поведение распределённых систем.

Ключевые преимущества OpenTelemetry:
- Унифицированные API и инструменты без привязки к конкретному поставщику
- Поддержка множества языков программирования и бэкенд-систем
- Сбор данных с низкими накладными расходами
- Надёжная поддержка распределённой трассировки

Предоставляет промежуточное ПО с поддержкой OpenTelemetry. Вы можете обратиться к [примеру](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) в официальном репозитории.
