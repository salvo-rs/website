## Cos'è SSE

Server-Sent Events (SSE) è una tecnologia web che consente al server di inviare dati in push al client. A differenza di WebSocket, SSE è una comunicazione unidirezionale: i dati possono essere inviati solo dal server al client, ideale per scenari in cui è necessario un aggiornamento in tempo reale dal server senza che il client debba inviare frequentemente dati. SSE si basa sul protocollo HTTP, è semplice da implementare e include un meccanismo automatico di riconnessione.

Fornisce un middleware per il supporto a `SSE`.

## Integrazione con modelli linguistici avanzati

Quando si utilizzano SDK come OpenAI o altri SDK per modelli linguistici avanzati per integrare l'IA, la restituzione dei risultati in streaming (Stream) è uno scenario comune. SSE offre una soluzione perfetta, consentendo di inviare in tempo reale i contenuti generati dall'IA al frontend, migliorando l'esperienza utente.

Ad esempio, quando si utilizza l'API Chat Completion di OpenAI con risposta in streaming, la funzionalità SSE di Salvo può essere sfruttata per inviare progressivamente i contenuti generati dall'IA al client, evitando che l'utente debba attendere il completamento della risposta.
