## Что такое SSE

Server-Sent Events (SSE) — это веб-технология, позволяющая серверам отправлять данные клиентам. В отличие от WebSocket, SSE является односторонним методом связи, где данные могут передаваться только с сервера на клиент. Она подходит для сценариев, когда серверу необходимо отправлять данные в реальном времени, но клиенту не требуется часто отправлять данные. SSE основана на протоколе HTTP, проста в реализации и имеет встроенные механизмы переподключения.

## Интеграция с большими языковыми моделями

При интеграции искусственного интеллекта с использованием OpenAI SDK или других SDK больших языковых моделей потоковые ответы (Stream) являются распространённым требованием. SSE предоставляет идеальное решение, позволяя доставлять контент, сгенерированный ИИ, на фронтенд в реальном времени, что обеспечивает лучший пользовательский опыт.

Например, при использовании OpenAI Chat Completion API с потоковыми ответами можно задействовать функциональность SSE в Salvo для постепенной отправки клиенту контента, сгенерированного ИИ, исключая задержку ожидания полного ответа.
