# JWT 驗證

JWT (JSON Web Token) 是一種開放標準 (RFC 7519)，用於在各方之間安全地傳輸資訊。它是一種緊湊的、URL安全的方式，以JSON物件形式表示聲明，這些聲明通常用於身份驗證和資訊交換。

JWT由三部分組成：
- **頭部(Header)** - 指定令牌類型和使用的簽名演算法
- **負載(Payload)** - 包含聲明(claims)，如用戶ID、角色、過期時間等
- **簽名(Signature)** - 用於驗證訊息在傳輸過程中沒有被更改

JWT在身份驗證流程中的典型使用方式：
1. 用戶登入後，伺服器產生JWT令牌
2. 令牌返回給客戶端並儲存（通常在localStorage或cookie中）
3. 後續請求中，客戶端在Authorization頭部攜帶此令牌
4. 伺服器驗證令牌的有效性並授權存取
