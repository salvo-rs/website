## ¿Qué es SSE?

Server-Sent Events (SSE) es una tecnología web que permite al servidor enviar datos al cliente de forma automática. A diferencia de WebSocket, SSE es unidireccional: solo transmite datos del servidor al cliente, ideal para escenarios donde se requiere que el servidor envíe datos en tiempo real sin necesidad de que el cliente los solicite frecuentemente. SSE se basa en el protocolo HTTP, es sencillo de implementar y cuenta con mecanismos de reconexión automática.

## Integración con modelos de lenguaje avanzados

Al utilizar SDKs de OpenAI u otros SDKs de modelos de lenguaje avanzado para integrar IA, la devolución de resultados en flujo continuo (Stream) es un escenario común. SSE ofrece una solución perfecta, permitiendo enviar el contenido generado por la IA al frontend en tiempo real, mejorando así la experiencia del usuario.

Por ejemplo, al integrar la API de Chat Completion de OpenAI con respuestas en flujo continuo, se puede aprovechar la funcionalidad SSE de Salvo para enviar progresivamente el contenido generado por la IA al cliente, evitando que el usuario perciba demoras mientras espera la respuesta completa.
