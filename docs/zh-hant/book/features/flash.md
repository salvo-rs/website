# Flash

提供 Flash Message 的功能的中間件.

`FlashStore` 提供對數據的存取操作. `CookieStore` 會在 `Cookie` 中存儲數據. 而 `SessionStore` 把數據存儲在 `Session` 中, `SessionStore` 必須和 `session` 功能一起使用.
