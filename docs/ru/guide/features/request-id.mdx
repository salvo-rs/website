# ID цепочки запроса

import { Tab, Tabs } from '@rspress/core/theme';

Промежуточное ПО Request Id является достаточно гибким. Генератор (IdGenerator) используется для создания ID, и вы можете определить собственный генератор ID, реализовав трейт `IdGenerator`. Генератор по умолчанию — `UlidGenerator`.

Кроме того, вы можете контролировать, перезаписывать ли уже существующий `requestid`. Также можно настроить `header_name` и другие параметры. Подробнее см. в [документации](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
