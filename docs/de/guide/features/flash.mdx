# Flash

Middleware zur Bereitstellung von Flash-Message-Funktionalität.

`FlashStore` ermöglicht Lese- und Schreibzugriff auf Daten. `CookieStore` speichert Daten in `Cookies`. `SessionStore` hingegen speichert Daten in der `Session` und muss in Verbindung mit der `session`-Funktionalität verwendet werden.
