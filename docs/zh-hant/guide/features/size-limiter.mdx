# 上傳檔案大小限制中介軟體

import { Tab, Tabs } from '@rspress/core/theme';

提供對請求上傳檔案大小限制的中介軟體功能。

## 應用場景

檔案大小限制器主要適用於以下情境：
- 防止使用者上傳過大檔案佔用伺服器資源
- 避免惡意攻擊者透過上傳大量資料導致伺服器記憶體溢位
- 為不同的 API 端點設定不同的請求大小限制策略
- 提升伺服器處理請求的效能與穩定性
