# Integración de OpenTelemetry

OpenTelemetry es un marco de observabilidad de código abierto que ofrece un enfoque estandarizado para recopilar y exportar datos de telemetría de aplicaciones, incluyendo trazas, métricas y registros. Ayuda a los desarrolladores a monitorear el rendimiento de las aplicaciones, solucionar problemas y comprender el comportamiento de los sistemas distribuidos.

Ventajas clave de OpenTelemetry:
- APIs y herramientas unificadas sin dependencia de proveedores
- Soporte para múltiples lenguajes de programación y sistemas backend
- Recopilación de datos con baja sobrecarga
- Soporte robusto para trazas distribuidas

Proporciona middleware con soporte para OpenTelemetry. Puedes consultar el [ejemplo](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) en el repositorio oficial.
