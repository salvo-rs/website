Middleware для ограничения размера загружаемых файлов в запросах.

## Сценарии использования

Size Limiter применяется в следующих случаях:
- Предотвращение загрузки слишком больших файлов, которые могут занять ресурсы сервера
- Защита от злоумышленников, пытающихся вызвать переполнение памяти сервера путем загрузки больших объемов данных
- Установка различных ограничений на размер запросов для разных API-эндпоинтов
- Повышение эффективности и стабильности обработки запросов сервером
