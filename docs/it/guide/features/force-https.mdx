# Forzare HTTPS

Il middleware `force-https` può reindirizzare tutte le richieste per utilizzare il protocollo HTTPS.

Se applicato a un `Router`, questo middleware forzerà il cambio di protocollo solo quando la rotta corrisponde; se la pagina non esiste, non verrà effettuato alcun reindirizzamento.

Tuttavia, è più comune voler reindirizzare automaticamente qualsiasi richiesta, anche quando la rotta non corrisponde e viene restituito un errore `404`. In questo caso, il middleware può essere aggiunto al `Service`. Indipendentemente dal fatto che la richiesta corrisponda a una rotta, il middleware aggiunto al `Service` verrà sempre eseguito.
