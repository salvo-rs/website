# Intégration OpenTelemetry

OpenTelemetry est un framework d'observabilité open source qui offre une approche standardisée pour collecter et exporter des données de télémétrie depuis les applications, incluant les traces, les métriques et les journaux. Il aide les développeurs à surveiller les performances des applications, résoudre les problèmes et comprendre le comportement des systèmes distribués.

Principaux avantages d'OpenTelemetry :
- API et outils unifiés sans dépendance à un fournisseur spécifique
- Prise en charge de multiples langages de programmation et systèmes backend
- Collecte de données à faible impact sur les performances
- Support robuste pour le traçage distribué

Fournit un middleware avec prise en charge d'OpenTelemetry. Vous pouvez consulter l'[exemple](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) dans le dépôt officiel.
