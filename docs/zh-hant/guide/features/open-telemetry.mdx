# OpenTelemetry 整合

OpenTelemetry 是一套開源的可觀測性框架，提供標準化的方式來收集與匯出應用程式的遙測資料（包含追蹤、指標與日誌）。它能協助開發者監控應用效能、診斷問題，並理解分散式系統的行為模式。

OpenTelemetry 的核心優勢：
- 統一的 API 與工具，無廠商鎖定限制
- 支援多種程式語言與後端系統
- 低負載的資料收集機制
- 對分散式追蹤的強大支援能力

提供支援 OpenTelemetry 的中介軟體。可參閱官方函式庫中的[範例程式](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger)。
