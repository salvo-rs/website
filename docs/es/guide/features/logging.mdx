# Middleware de Registro

El registro de logs desempeña un papel crucial en las aplicaciones web, ya que permite:
- Proporcionar información detallada sobre el procesamiento de solicitudes, ayudando a los desarrolladores a rastrear el comportamiento de la aplicación
- Facilitar la resolución de problemas y la depuración, especialmente en entornos de producción
- Supervisar el rendimiento de la aplicación y el uso de recursos
- Registrar patrones de acceso de usuarios y excepciones del sistema
- Cumplir con requisitos de auditoría de seguridad y normativas

Salvo ofrece un middleware con funcionalidad básica de registro. Si el middleware se agrega directamente a un `Router`, no podrá capturar los errores 404 que se devuelven cuando ninguna ruta coincide. Se recomienda agregarlo al `Service`.
