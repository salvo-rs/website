# Middleware de limitation de taille des fichiers téléchargés

import { Tab, Tabs } from '@rspress/core/theme';

Fournit un middleware pour limiter la taille des fichiers téléchargés dans les requêtes.

## Cas d'utilisation

Size Limiter est principalement utilisé dans les scénarios suivants :
- Empêcher les utilisateurs de télécharger des fichiers trop volumineux qui consommeraient les ressources du serveur
- Éviter que des attaquants malveillants ne provoquent un débordement de mémoire du serveur en téléchargeant de grandes quantités de données
- Définir des stratégies de limitation de taille de requête différentes pour différents points de terminaison d'API
- Améliorer l'efficacité et la stabilité du traitement des requêtes par le serveur
