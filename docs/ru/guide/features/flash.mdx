# Flash

Промежуточное ПО, предоставляющее функциональность Flash-сообщений.

`FlashStore` обеспечивает операции чтения и записи данных. `CookieStore` хранит данные в `Cookie`. В то время как `SessionStore` сохраняет данные в `Session`, и его использование обязательно совместно с функцией `session`.
