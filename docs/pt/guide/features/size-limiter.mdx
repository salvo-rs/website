# Middleware de Limitação de Tamanho de Upload de Arquivos

import { Tab, Tabs } from '@rspress/core/theme';

Fornece um middleware para limitar o tamanho de arquivos enviados em requisições.

## Cenários de Aplicação

O Size Limiter é utilizado principalmente nos seguintes cenários:
- Impedir que usuários enviem arquivos excessivamente grandes, ocupando recursos do servidor
- Evitar que atacantes mal-intencionados causem estouro de memória do servidor através do envio massivo de dados
- Definir políticas de limite de tamanho de requisição diferentes para diferentes endpoints de API
- Melhorar a eficiência e estabilidade do processamento de requisições pelo servidor
