## Введение в WebSocket

WebSocket — это протокол, обеспечивающий полнодуплексную связь через одно TCP-соединение. Он упрощает обмен данными между клиентом и сервером, позволяя серверу активно отправлять данные клиенту. В протоколе WebSocket браузеру и серверу достаточно выполнить рукопожатие один раз, чтобы установить постоянное соединение и обеспечить двустороннюю передачу данных.

В отличие от традиционных HTTP-запросов, соединение WebSocket остаётся открытым после установки, что позволяет избежать частого создания и разрыва соединений, значительно снижая накладные расходы на связь. Это делает WebSocket особенно подходящим для приложений реального времени, таких как онлайн-игры, чат-приложения, биржевые тикеры и другие приложения, требующие обновлений в реальном времени.

Salvo предоставляет лаконичный и мощный API через своё промежуточное ПО WebSocket, позволяя разработчикам легко интегрировать функциональность WebSocket в свои приложения.
