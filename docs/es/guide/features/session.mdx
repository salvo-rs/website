# Sesión

import { Tab, Tabs } from '@rspress/core/theme';

Middleware que proporciona soporte para `Session`.

## ¿Qué es una Sesión?

Una sesión es un conjunto de datos de usuario almacenados en el servidor, comúnmente utilizados para rastrear el estado del usuario y guardar datos temporales. En aplicaciones web, las sesiones resuelven el problema de la falta de estado en el protocolo HTTP, permitiendo al servidor identificar múltiples solicitudes provenientes del mismo usuario.

## Uso de Sesión en Salvo

En Salvo, el middleware de sesión ofrece una API sencilla para leer y escribir datos de sesión. A través de `SessionDepot` se puede obtener una instancia de sesión, y mediante métodos como `get`, `set` y `remove` se pueden gestionar los datos de sesión.

## Usos Comunes

- Autenticación de usuarios y gestión del estado de inicio de sesión
- Almacenamiento temporal de datos de formularios
- Guardado de preferencias del usuario
