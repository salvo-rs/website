# Промежуточное ПО для ведения журналов

Ведение журналов играет ключевую роль в веб-приложениях, позволяя:
- Предоставлять детальную информацию о процессе обработки запросов, помогая разработчикам отслеживать поведение приложения
- Содействовать выявлению проблем и отладке, особенно в производственной среде
- Мониторить производительность приложения и использование ресурсов
- Регистрировать шаблоны доступа пользователей и системные исключения
- Соответствовать требованиям аудита безопасности и нормативным стандартам

Salvo предоставляет базовое промежуточное ПО с функцией ведения журналов. Если промежуточное ПО добавлено непосредственно в `Router`, оно не сможет перехватывать ошибки 404, возвращаемые при отсутствии совпадений со всеми `Router`. Рекомендуется добавлять его в `Service`.
