# Принудительное использование HTTPS

Промежуточное ПО `force-https` позволяет принудительно перенаправлять все запросы на использование протокола HTTPS.

Если это промежуточное ПО применяется к `Router`, принудительное изменение протокола происходит только при совпадении маршрута. В случае отсутствия запрашиваемой страницы перенаправление не выполняется.

Однако чаще возникает необходимость автоматического перенаправления любых запросов, даже когда маршрут не найден и возвращается ошибка `404`. В таких случаях промежуточное ПО можно добавить к `Service`. Промежуточное ПО, добавленное к `Service`, всегда выполняется независимо от того, был ли запрос успешно сопоставлен с маршрутом.
