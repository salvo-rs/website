# Forzar HTTPS

El middleware `force-https` puede redirigir todas las solicitudes para que utilicen el protocolo HTTPS.

Si este middleware se aplica a un `Router`, solo forzará el cambio de protocolo cuando coincida la ruta. Si la página no existe, no se realizará la redirección.

Sin embargo, es más común querer que cualquier solicitud se redirija automáticamente, incluso cuando la ruta no coincide y se devuelve un error `404`. En este caso, se puede agregar el middleware al `Service`. Independientemente de si la solicitud coincide con una ruta exitosamente, el middleware agregado al `Service` siempre se ejecutará.
