# Middleware de Tempo Limite

Fornece um middleware para suporte a tempo limite.

## Cenários de Uso

Em serviços web, algumas requisições podem demorar muito para serem processadas devido a vários motivos (como consultas lentas ao banco de dados, respostas lentas de serviços externos, etc.). Para evitar que essas requisições de longa duração ocupem recursos do servidor e afetem o processamento de outras requisições, pode-se usar o middleware de tempo limite para definir um tempo máximo de processamento. Se esse limite for excedido, a requisição será automaticamente interrompida e retornará um erro de tempo limite.

Através de `Timeout::new(Duration::from_secs(5))`, definimos um limite de tempo de 5 segundos para todas as requisições.
