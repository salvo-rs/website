# Перехват паник в запросах

Catch Panic используется для перехвата сбоев, возникающих во время обработки запросов в программе. Подробные API смотрите в [документации](https://docs.rs/salvo_extra/latest/salvo_extra/catch_panic/index.html).

> **Примечание**: Для использования `CatchPanic` необходимо подключить функцию `catch-panic` в `Cargo.toml`:
> ```toml
> salvo = { version = "xxx", features = ["catch-panic"] }
> ```

## Введение в промежуточное ПО

`CatchPanic` — это промежуточное ПО, предназначенное для перехвата паник в обработчиках. При возникновении паники во время обработки запроса оно перехватывает её и записывает ошибку 500 Internal Server Error в ответ, предотвращая аварийное завершение всего сервера.

**Важное замечание**: Это промежуточное ПО должно использоваться первым, чтобы гарантировать перехват паник из других промежуточных ПО или обработчиков.

## Базовое использование

```rust
use salvo_core::prelude::*;
use salvo_extra::catch_panic::CatchPanic;

#[handler]
async fn hello() {
    panic!("panic error!");
}

#[tokio::main]
async fn main() {
    let router = Router::new().hoop(CatchPanic::new()).get(hello);
    let acceptor = TcpListener::new("0.0.0.0:5800").bind().await;
    Server::new(acceptor).serve(router).await;
}
```

## Сравнение с другими фреймворками для быстрого понимания концепции

### Axum

Аналогично промежуточному ПО `catch_panic` в `Tower` от Axum:

```rust
use axum::{
    Router,
    routing::get,
    http::StatusCode,
};
use tower::ServiceBuilder;
use tower_http::catch_panic::CatchPanicLayer;

async fn panic_handler() -> &'static str {
    panic!("panic error!");
}

#[tokio::main]
async fn main() {
    let app = Router::new()
        .route("/", get(panic_handler))
        .layer(
            ServiceBuilder::new()
                .layer(CatchPanicLayer::new())
        );
    
    axum::Server::bind(&"0.0.0.0:3000".parse().unwrap())
        .serve(app.into_make_service())
        .await
        .unwrap();
}
```

### Gin (Go)

В фреймворке Gin для языка Go аналогично промежуточному ПО Recovery:

```go
package main

import (
    "github.com/gin-gonic/gin"
)

func main() {
    r := gin.Default() // По умолчанию включает промежуточное ПО Recovery
    
    r.GET("/", func(c *gin.Context) {
        panic("panic error!")
    })
    
    r.Run(":8080")
}
```
