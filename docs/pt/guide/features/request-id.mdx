# ID da Cadeia de Requisição

import { Tab, Tabs } from '@rspress/core/theme';

O middleware de ID de requisição é altamente flexível. O gerador de ID (IdGenerator) é utilizado para criar identificadores, e você pode definir seu próprio gerador desde que implemente a trait `IdGenerator`. O gerador padrão fornecido é o `UlidGenerator`.

Além disso, você pode controlar se deseja sobrescrever um `requestid` existente. Também é possível configurar `header_name` e outras opções. Para mais detalhes, [consulte a documentação](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
