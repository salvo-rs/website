# Flash

Middleware que fornece a funcionalidade de Mensagens Flash.

`FlashStore` oferece operações de armazenamento e recuperação de dados. `CookieStore` armazena os dados em `Cookies`, enquanto `SessionStore` armazena os dados na `Sessão`. O `SessionStore` deve ser usado em conjunto com a funcionalidade de `sessão`.
