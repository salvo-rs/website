# OpenTelemetry集成

OpenTelemetry 是一个开源的可观测性框架，提供了标准化的方式来收集和导出应用程序的遥测数据（包括跟踪、指标和日志）。它能帮助开发者监控应用性能、排查问题并了解分布式系统的行为。

OpenTelemetry 的主要优势:
- 统一的 API 和工具，无厂商锁定
- 支持多种编程语言和后端系统
- 低开销的数据收集
- 对分布式追踪的强大支持

提供对 OpenTelemetry 支持的中间件. 可以参考官方库中的[示例](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger).
