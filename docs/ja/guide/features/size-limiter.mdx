リクエストのアップロードファイルサイズを制限するミドルウェアを提供します。

## 適用シナリオ

Size Limiterは主に以下のような場面で使用されます：
- ユーザーが過大なファイルをアップロードしてサーバーリソースを占有するのを防止
- 悪意のある攻撃者が大量データをアップロードしてサーバーのメモリオーバーフローを引き起こすのを回避
- 異なるAPIエンドポイントごとに異なるリクエストサイズ制限ポリシーを設定
- サーバーのリクエスト処理効率と安定性の向上
