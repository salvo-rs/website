# 超時中介軟體

提供對超時支援的中介軟體。

## 使用場景

在 Web 服務中，某些請求可能因為各種原因（如資料庫查詢緩慢、外部服務回應慢等）導致處理時間過長。為避免這些長時間執行的請求佔用伺服器資源，影響其他請求的處理，可以使用超時中介軟體為請求設定最大處理時間限制。超過這個時間限制，請求將被自動中斷，並回傳超時錯誤。
