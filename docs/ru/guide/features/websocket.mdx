## Введение в WebSocket

WebSocket — это протокол для полнодуплексной связи по одному TCP-соединению. Он упрощает обмен данными между клиентом и сервером, позволяя серверу активно отправлять данные клиенту. В протоколе WebSocket браузеру и серверу достаточно выполнить однократное рукопожатие, чтобы установить постоянное соединение и осуществлять двустороннюю передачу данных.

В отличие от традиционных HTTP-запросов, соединение WebSocket остаётся открытым после установления, что исключает частое создание и разрыв соединений и значительно снижает коммуникационные издержки. Это делает WebSocket особенно подходящим для приложений реального времени, таких как онлайн-игры, чат-приложения, биржевые котировки и другие программы, требующие мгновенного обновления данных.

Salvo предоставляет простой и мощный API через своё промежуточное ПО WebSocket, позволяя разработчикам легко интегрировать функциональность WebSocket в приложения.
