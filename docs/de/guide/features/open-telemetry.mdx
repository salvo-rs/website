# OpenTelemetry-Integration

OpenTelemetry ist ein Open-Source-Observability-Framework, das einen standardisierten Ansatz zur Erfassung und zum Export von Telemetriedaten aus Anwendungen bietet, einschließlich Traces, Metriken und Logs. Es unterstützt Entwickler dabei, die Anwendungsleistung zu überwachen, Probleme zu beheben und das Verhalten verteilter Systeme zu verstehen.

Wesentliche Vorteile von OpenTelemetry:
- Vereinheitlichte APIs und Tools ohne Vendor-Lock-in
- Unterstützung für mehrere Programmiersprachen und Backend-Systeme
- Ressourcenschonende Datenerfassung
- Umfassende Unterstützung für verteiltes Tracing

Bietet Middleware mit OpenTelemetry-Unterstützung. Weitere Informationen finden Sie im [Beispiel](https://github.com/salvo-rs/salvo/tree/main/examples/otel-jaeger) im offiziellen Repository.
