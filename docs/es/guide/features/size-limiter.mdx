Proporciona un middleware para limitar el tamaño de los archivos subidos en las solicitudes.

## Casos de uso

El Limitador de Tamaño (Size Limiter) se utiliza principalmente en los siguientes escenarios:
- Evitar que los usuarios suban archivos demasiado grandes que consuman recursos del servidor
- Prevenir ataques maliciosos que intenten saturar la memoria del servidor con grandes volúmenes de datos
- Establecer diferentes políticas de límite de tamaño para distintos endpoints de API
- Mejorar la eficiencia y estabilidad del servidor al procesar solicitudes
