# Identifiant de Chaîne de Requête

import { Tab, Tabs } from '@rspress/core/theme';

Le middleware d'identifiant de requête est extrêmement flexible. Le générateur d'identifiants (IdGenerator) est utilisé pour générer les ID, et vous pouvez définir votre propre générateur d'identifiants tant qu'il implémente le trait `IdGenerator`. Le générateur par défaut fourni est `UlidGenerator`.

De plus, vous pouvez contrôler s'il faut écraser un `requestid` existant. Vous pouvez également configurer `header_name` et d'autres paramètres. Pour plus de détails, veuillez [consulter la documentation](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
