## O que é SSE

Server-Sent Events (SSE) é uma tecnologia web que permite que um servidor envie dados por push para um cliente. Diferente do WebSocket, o SSE é unidirecional - apenas o servidor pode enviar dados para o cliente, sendo ideal para cenários onde é necessário que o servidor envie dados em tempo real sem que o cliente precise enviar informações frequentemente. O SSE é baseado no protocolo HTTP, tem implementação simples e possui mecanismo de reconexão automática.

Oferece um middleware com suporte a `SSE`.

## Integração com modelos de linguagem grandes

Ao utilizar SDKs como o OpenAI ou outros SDKs de modelos de linguagem grandes para integrar IA, o retorno de resultados em fluxo contínuo (Stream) é um cenário de demanda comum. O SSE fornece uma solução perfeita, permitindo que o conteúdo gerado pela IA seja enviado em tempo real para o front-end, proporcionando uma melhor experiência do usuário.

Por exemplo, ao integrar a API de Chat Completion da OpenAI com resposta em fluxo, é possível utilizar a funcionalidade SSE do Salvo para enviar gradualmente o conteúdo gerado pela IA para o cliente, evitando a sensação de latência causada pela espera da resposta completa.
