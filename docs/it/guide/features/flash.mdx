# Flash

Middleware che fornisce la funzionalità di Flash Message.

`FlashStore` offre operazioni di accesso e memorizzazione dei dati. `CookieStore` archivia i dati nei `Cookie`, mentre `SessionStore` li memorizza nella `Session`. `SessionStore` deve essere utilizzato insieme alla funzionalità `session`.
