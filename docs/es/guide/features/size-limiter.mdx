# Middleware de Límite de Tamaño de Archivos Subidos

import { Tab, Tabs } from '@rspress/core/theme';

Proporciona middleware para limitar el tamaño de archivos subidos en las solicitudes.

## Casos de Uso

Size Limiter se utiliza principalmente en los siguientes escenarios:
- Evitar que los usuarios suban archivos demasiado grandes que consuman recursos del servidor
- Prevenir ataques maliciosos que puedan causar desbordamiento de memoria del servidor mediante la subida de grandes cantidades de datos
- Establecer diferentes políticas de límite de tamaño de solicitud para distintos endpoints de API
- Mejorar la eficiencia y estabilidad del servidor al procesar solicitudes
