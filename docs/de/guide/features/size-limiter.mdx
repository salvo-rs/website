# Middleware zur Begrenzung der Datei-Upload-Größe

import { Tab, Tabs } from '@rspress/core/theme';

Eine Middleware zur Begrenzung der Größe hochgeladener Dateien in Anfragen.

## Anwendungsfälle

Die Größenbegrenzung wird hauptsächlich in folgenden Szenarien eingesetzt:
- Verhindern, dass Benutzer übermäßig große Dateien hochladen, die Serverressourcen verbrauchen
- Vermeiden von Server-Speicherüberlauf durch böswillige Angreifer, die große Datenmengen hochladen
- Festlegen unterschiedlicher Richtlinien zur Anfragegrößenbegrenzung für verschiedene API-Endpunkte
- Verbesserung der Effizienz und Stabilität der Serveranfrageverarbeitung
