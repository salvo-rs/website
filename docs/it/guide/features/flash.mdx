# Flash

Middleware che fornisce la funzionalità di Flash Message.

`FlashStore` offre operazioni di lettura e scrittura dei dati. `CookieStore` memorizza i dati nei `Cookie`. Mentre `SessionStore` salva i dati nella `Session` e deve essere utilizzato insieme alla funzionalità `session`.
