# Flash

Промежуточное ПО для функциональности Flash Message.

`FlashStore` предоставляет операции чтения и записи данных. `CookieStore` хранит данные в `Cookie`, тогда как `SessionStore` сохраняет данные в `Session`. Для работы `SessionStore` необходимо использовать вместе с функцией `session`.
