# Flash

Middleware that provides Flash Message functionality.

`FlashStore` provides operations for storing and retrieving data. `CookieStore` stores data in `Cookie`s. `SessionStore`, on the other hand, stores data in the `Session`. `SessionStore` must be used in conjunction with the `session` feature.
