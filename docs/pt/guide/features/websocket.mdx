## Introdução ao WebSocket

WebSocket é um protocolo de comunicação full-duplex sobre uma única conexão TCP. Ele simplifica a troca de dados entre cliente e servidor, permitindo que o servidor envie dados ativamente ao cliente. No protocolo WebSocket, o navegador e o servidor só precisam realizar um handshake uma vez para estabelecer uma conexão persistente e realizar transmissão bidirecional de dados.

Diferente das requisições HTTP tradicionais, uma conexão WebSocket permanece aberta após ser estabelecida, evitando a criação e encerramento frequentes de conexões, o que reduz significativamente a sobrecarga de comunicação. Isso torna o WebSocket especialmente adequado para aplicações em tempo real, como jogos online, aplicativos de chat, cotações de ações e outros programas que exigem atualizações em tempo real.

O Salvo fornece uma API concisa e poderosa através de seu middleware WebSocket, permitindo que desenvolvedores integrem facilmente funcionalidades WebSocket em suas aplicações.
