## O que é SSE

Server-Sent Events (SSE) é uma tecnologia web que permite que um servidor envie dados para um cliente de forma ativa. Diferente do WebSocket, o SSE é uma comunicação unidirecional, onde os dados são enviados apenas do servidor para o cliente, sendo ideal para cenários em que o servidor precisa enviar dados em tempo real sem que o cliente precise enviar informações frequentemente. O SSE é baseado no protocolo HTTP, de implementação simples, e possui um mecanismo de reconexão automática em caso de queda de conexão.

## Integração com Modelos de Linguagem de Grande Escala

Ao integrar IA usando SDKs como o OpenAI ou outros SDKs de modelos de linguagem de grande escala, o retorno de resultados em fluxo contínuo (Stream) é um cenário de demanda comum. O SSE oferece uma solução perfeita, permitindo que o conteúdo gerado pela IA seja enviado em tempo real para o front-end, proporcionando uma melhor experiência do usuário.

Por exemplo, ao utilizar a API de Chat Completion da OpenAI com resposta em fluxo, é possível aproveitar a funcionalidade SSE do Salvo para enviar gradualmente o conteúdo gerado pela IA para o cliente, evitando a sensação de espera pelo retorno completo.
