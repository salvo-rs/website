# 強制 HTTPS

`force-https` 中介軟體能夠強制所有請求轉向使用 HTTPS 通訊協定。

若將此中介軟體應用於 `Router`，僅在路由匹配成功時才會強制轉換通訊協定；若遇到頁面不存在的情況，則不會進行轉向。

然而更常見的需求是希望所有請求都能自動轉向，即使在路由未能匹配而返回 `404` 錯誤時亦然。此時可將中介軟體添加至 `Service` 層級。無論請求是否成功匹配路由，添加於 `Service` 的中介軟體皆會執行。
