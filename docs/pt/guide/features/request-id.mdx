# ID do Rastreamento de Solicitação

import { Tab, Tabs } from '@rspress/core/theme';

O middleware de Request Id é bastante flexível. O gerador (IdGenerator) é usado para criar IDs, e você pode definir seu próprio gerador de IDs, desde que implemente o trait `IdGenerator`. O gerador padrão fornecido é o `UlidGenerator`.

Além disso, você pode controlar se deseja sobrescrever um `requestid` já existente. Também é possível configurar `header_name` e outros parâmetros. Para mais detalhes, consulte a [documentação](https://docs.rs/salvo_extra/latest/salvo_extra/request_id/index.html).
